    SimulationLayout(numOfBS, numOfUE, heightOfBS, heightOfUE, ISD, layoutType = "Hexagonal", ueDropMethod = None, UEdistibution = None, UEheightDistribution = "random", numOfSectorsPerSite = 1, minUEBSDistance = 0, layoutWidth = None, layoutLength = None, bsRoute = None, ueRoute = None, useInitueLocations = False, radiusForCircularUEDrop = None)

    A class for generating Simulation Layout.    
    Support important features such as computing Line of Sight (LOS) probability for different scenario or terrain.   
    It can also display the 2D plot of simulation layout using :meth:`display2DTopology()` method. 

    
    Example
    --------
    >>> simLayoutObj = SimulationLayout(numOfBS = 19,
    ...                                 numOfUE = 100,
    ...                                 heightOfBS = 25,
    ...                                 heightOfUE = 10, 
    ...                                 ISD = 500, 
    ...                                 layoutType = "Hexagonal",
    ...                                 numOfSectorsPerSite = 3,
    ...                                 minUEBSDistance = 35)     
    >>> simLayoutObj(terrain = "UMa", carrierFreq = 5) 
    
    Above code snippet shows basic instantiation of :class:`SimulationLayout` class with 3 sector "Hexagonal" 
    layout having 19 BSs, 100 UEs with an ISD of 500 meters and with a carrierFreq of 5 GHz for the terrain "UMa".
    More examples can be found in the sections below.

    
    Parameters
    -----------      
    numOfBS : int or NumPy.ndarray of int
        Number of Base Stations(BSs) in simulation. Must be a positive int or array of positive int.
    numOfUE : int
        Number of User Equipment(UE) in simulation. Must be positive.
    heightOfBS : int or float
        height of each BS in meters. Must be a positive number.
    heightOfUE : int or float
        height of each UE in meters. Must be a positive number.
    ISD : int or float
        InterSite Distance. Defined as distance between neighbouring sites. Must be positive. 
    layoutType : str
        Specifies the geometry of placement of all the  BSs.
        Defaults to "Hexagonal". Supported ``layoutType`` are "Rectangular"  and "Hexagonal" 
    ueDropMethod : None or str
        Specifies the geometry of UE drops.    
        Defaults to None. In the default case it takes the same geometry as BS ``layoutType``        
        Supported ``ueDropMethod`` are  "Rectangular" , "Hexagonal" and "Circular"
    UEdistribution : None or str
        Specify the way in which the UEs are distributed for a chosen ``ueDropMethod``
        Defaults to None. In the default case it takes the value "random". i.e., all the UEs are dropped randomly in the choosen geometry.

            - If ``ueDropMethod`` is "Hexagonal" then supported ``UEdistibution`` are "random" or "equal".

                - For "random", UEs are distributed randomly anywhere in the hexagonal grid connecting all the edges of the outermost hexagons.
                - For "equal",  number of UEs distributed in each hexagon of a grid are equal.
            - If ``ueDropMethod`` is "Rectangular" then supported ``UEdistibution`` are "random" or "uniform".

                * For "random", UEs are distributed randomly anywhere inside the rectangle.
                * For "uniform", UEs are distributed as a grid of uniformly distributed points inside a rectangle.
            - If ``ueDropMethod`` is "Circular" then supported ``UEdistibution`` is "random".
              For "random", UEs are distributed randomly anywhere inside a circle.
    UEheightDistribution: str
        Specify how the heights of UE are distributed in the simulation.
        
            - Defaults to "random", means UE heights can take any random value between 0 to ``heightOfUE``
            - Can also take "equal", means all the UE heights are of the same. i.e., ``heightOfUE``
    numOfSectorsPerSite: int
        Number of sectors per site. Must be a positive integer. Defaults to 1.
    minUEBSDistance: int or float
        minimum distance between BS and UE. Must be positive and less that half of ``ISD``
    layoutWidth: None or int or float
        Width of the BS layout. Must be Positive. Defaults to None.

            - In the default case it takes the value of ``ISD`` for "Hexagonal" layout 
            - ``ISD`` * (`num of BS along columns`) for "Rectangular" layout.  
    layoutLength: None or int or float
        Length of the BS layout. Must be Positive. Defaults to None.

            - In the default case it takes the value of ``ISD`` for "Hexagonal" layout.
            - ``ISD`` * (`num of BS along rows`) for "Rectangular" layout. 
    bsRoute: None or NodeMobility
        Route or location of all the BSs across different snapshots.
        Defaults to None. Must be a single or array of object(s) of the class ``NodeMobility``
    ueRoute: None or NodeMobility
        Route or location of all the UEs across different snapshots.
        Defaults to None. Must be a single or array of object(s) of the class ``NodeMobility``
    useInitueLocations: bool
        A boolen variable or flag specifying whether initial UE locations of the SimulationLayout class are the same as initial UE locations generated at ``NodeMobility`` class or not.
        Defaults to False, means initial UE locations of ``SimulationLayout`` class are not based on initial UE locations of ``NodeMobility`` class.
    radiusForCircularUEDrop: None or int or float
        Radius of circular ``ueDropMethod`` . Must be positive. Defaults to None.

            - In the default case it takes the value of ``ISD`` for "Hexagonal" layout 
            - 0.5 * ``ISD`` * (`num of BS along rows + num of BS along columns`) for "Rectangular" layout.


    .. important::
       If ``layoutType`` is "Rectangular" then ``numOfBS`` must be a  `numpy array of shape (2,)`
       i.e., ``numOfBS`` =  `np.array([ num of BS along rows, num of BS along columns ])`


    Input
    ------
        terrain : str or NumPy.ndarray of str
            Propagation scenario/terrain for fast fading Spatial Channel Model (SCM) from section 7.5 of TR 38.901. When passed as a numpy array it takes a shape of (3,).
            i.e., array of 3 terrain each correspond to BS-UE, UE-UE and BS-BS links respectively.

            Current implementation support the following scenario or ``terrain`` 

                * Rural Macro "RMa"
                * Urban Macro "UMa" 
                * Urban Micro "UMi" 
                * Indoor Hotspot - Open Office  "InH-OO" 
                * Indoor Hotspot - Mixed Office "InH-MO" 
                * Indoor Factory - Sparse clutter Low BS "InF-SL"
                * Indoor Factory - Sparse clutter High BS "InF-SH"
                * Indoor Factory - Dense clutter Low BS "InF-DL"
                * Indoor Factory - Dense clutter High BS "InF-DH" 
                * Indoor Factory - High Tx High Rx "InF-HH"

            This variable also caputures the type of Tapped Delay Line (TDL) or Cluster Delay Line (CDL) model for link-level simulations.        
            Three CDL models, namely "CDL-A", "CDL-B" and "CDL-C" for NLOS while "CDL-D" and "CDL-E" are for LOS. Ref: section 7.7.1 of TR 38.901.
            Three TDL models, namely "TDL-A", "TDL-B" and "TDL-C" for NLOS while "TDL-D" and "TDL-E" are for LOS. Ref: section 7.7.1 of TR 38.901.                     
        carrierFreq : int or float or NumPy.ndarray of int or float
            Carrier Frequency in GHz. Must be positive.
            When passed as a numpy array it takes a dimension of 1 or shape (num of carrier frequencies, ) with all the entries taking positive ints or floats.
        ueAntennaArray : None or ``AntennaArrays`` or NumPy.ndarray of ``AntennaArrays``
            Antenna Array of every UE in the simulation. Defaults to None.

                - In the default case Antenna Array of all the UEs are having the following configuration
                  ``antennaType`` ="OMNI", ``arrayStructure`` = np.array([1,1,1,1,1], ``antennaSpacing`` = np.array([1, 1, 1, 1]).
                - When passed as a numpy array, the size has to match with the size of ``carrierFreq`` array. Otherwise unexpected results turn up. 
        bsAntennaArray : None or ``AntennaArrays`` or NumPy.ndarray of ``AntennaArrays``
            Antenna Array of every BS in the simulation. Defaults to None.

                - In the default case Antenna Array of all the BSs are having the following configuration
                  ``antennaType`` ="3GPP_38.901", ``arrayStructure`` = np.array([1,1,1,1,1], ``antennaSpacing`` = np.array([1, 1, 1, 1]).
                - When passed as a numpy array the size has to match with the size of ``carrierFreq`` array. Otherwise unexpected results turn up.
        indoorUEfraction: int or float
            Specifies the ratio of num of indoor UEs to the total num of UEs in the simulation. Defaults to 0.2 
            Note that every value must be positive and lie in the closed interval [0,1] as it is a fraction. 
        lengthOfIndoorObject: int or float
            Length of indoor object assuming the object geometry as a rectangle. Must be a positive number. Defaults to 1 meter.
        widthOfIndoorObject: int or float
            Width of indoor object assuming the object geometry as a rectangle. Must be a positive number. Defaults to 1 meter. 
        heightOfRoom: int or float
            Height of the Room or Ceiling. Must be a positive number taking values from the interval [minVal, maxVal].
            where minVal, maxVal are terrain dependent. Defaults to 3 meter. 

                - For Indoor Factory Dense scenario minVal = 5 and maxVal = 15.
                - For other Indoor Factory scenario minVal = 5 and maxVal = 25.
                - For all the other scenario minVal = 0 and maxVal = any positive int or float greater than 0.
                - Note that every value is assumed to be in meters.
        minNumberOfFloors: int
            Minimum number of floors. Must be a positive int greater than or equal to 0. Defaults to 1. Valid only for Indoor Scenario.
        maxNumberOfFloors: int
            Maximum number of floors. Must be a positive int greater than or equal to 0. Defaults to 1. Valid only for Indoor Scenario. 
        forceLOS: bool
            A boolean variable specifying whether every link state to be forced to LOS or not.
            Defaults to False correspond to every link state value is being generated by computation of probability of LOS.
            If true, then every link state value is forced to LOS with out invoking the computation of LOS probability.   
        is3GPPBaseline: bool
            A boolean variable specifying whether all the parameters or variables are in compliant with 3GPP or not.
            Defaults to true making all the variables 3GPP compliant.
            Current implementation support variables to take any value not necessarily restricted to 3GPP base line.        
        clutterHeight: int or float
            Height of the clutter in Indoor Factory scenario. Defaults to 5 meter.
            Must be a positive number and should not exceed minimum(10,heightOfRoom), where every value is assumed to be in meters.
        clutterSize: int or float
            Size of the clutter in Indoor Factory scenario. Must be a positive number.

                * If ``is3GPPBaseline`` true and if the scenario is Sparse, then it takes a value of 10, which is the default value.
                * If ``is3GPPBaseline`` true and if the scenario is Dense, then it takes a value of 2.
                * If ``is3GPPBaseline`` false then it must be a positive number from the interval [minVal, maxVal], where the maxVal should always be greater than or equal to the minVal.            
        clutterDensity: int or float
            Density of the clutter in Indoor Factory scenario. Must be a positive number lies in the range [minVal, maxVal]. Defaults to 0.4

                * If ``is3GPPBaseline`` true and the scenario is Dense takes a minVal = 0.4 and maxVal = 1
                * If ``is3GPPBaseline`` true and the scenario is Sparse takes a minVal = 0 and maxVal = 0.4
                * For all the other cases it must take minVal = 0 and maxVal = 1.
                * Note that all the values are assumed to be in meter. 

                              
    .. important::
       ``Supported terrain`` are 
        ``{"RMa", "UMa", "UMi", "InH-OO", "InH-MO", "InF-SL","InF-SH", "InF-DL", "InF-DH", "InF-HH", 
        "cdl-a" or "CDL-A", "cdl-b" or "CDL-B", "cdl-c" or "CDL-C", "cdl-d" or "CDL-D", "cdl-e" or "CDL-E",
        "tdl-a" or "TDL-A", "tdl-b" or "TDL-B", "tdl-c" or "TDL-C", "tdl-d" or "TDL-D", "tdl-e" or "TDL-E" }``.
        User can also access ``supportedTerrains`` attribute of the ::class::`SimulationLayout` to see the list of supported terrain.         

    Raises
    -------
    ValueError
        If ``numOfBS`` is neither a number nor a numpy array.
    ValueError
        If ``numOfUE`` is not an instance of int type/ an int type and not positive.
    ValueError
        If ``heightOfBS`` is not an instance of int type/ an int type and not positive.
    ValueError
        If ``heightOfUE`` is not an instance of int type/ an int type and not positive.   
    ValueError
        If ``ISD`` is not an instance of int/float type / an int/float type and not positive.
    ValueError
        If ``layoutType`` is not an instance of str type / a str and does not belong to the list ["Hexagonal", "Rectangular"] 
    ValueError
        If ``terrain`` is not an instance of str/ a str and not belong to the list of supported terrain 
            ["RMa", "UMa", "UMi", "InH-OO", "InH-MO", "InF-SL", "InF-SH", "InF-DL", "InF-DH", "InF-HH"]          
    ValueError
        If ``carrierFreq`` is neither a number nor a numpy array.
        If number and not an instance of int/float
        If numpy array and dimension other than 1 / elements of an array are not instances of int/float / elements of an array are either ints/floats but not positive.     
    """   
    def __init__(self, numOfBS, numOfUE, heightOfBS, heightOfUE, ISD, layoutType = "Hexagonal", 
                 ueDropMethod = None, UEdistibution = None, UEheightDistribution = "random", 
                 numOfSectorsPerSite = 1, minUEBSDistance = 0, layoutWidth = None, layoutLength = None, 
                 bsRoute = None, ueRoute = None, useInitueLocations = False, radiusForCircularUEDrop = None):
        
        # The  Constructor method for Simulation Layout Class.         
        
        self.numOfSectorsPerSite  =  numOfSectorsPerSite    # num of sectors per micro site
        self.layoutType           =  layoutType             # BS layout Type
        
        self.numOfBS              =  numOfBS                # numOfBS
        self.numOfUE              =  numOfUE                # numOfUE
        self.heightOfBS           =  heightOfBS             # height of BS
        self.heightOfUE           =  heightOfUE             # height of UE
        self.ISD                  =  ISD                    # Inter Site Distance   
        self.minUEBSDistance      =  minUEBSDistance        # Minimum distance between BS and UE

        self.ueDropMethod         =  ueDropMethod           # Method for dropping users

        self.UEdistibution        =  UEdistibution          # UE drop type
        self.UEheightDistribution =  UEheightDistribution   # UE height distribution
                
        self.bsRoute              =  bsRoute                # Route/Location where BS(s) is/are deployed
        self.ueRoute              =  ueRoute                # Route/Location where UE(s) is/are droped        
        

        #************************************************************************
        ## Placing the BSs based on the Layout type
        if (self.layoutType == "Rectangular"):

            self.layoutObj   = RectangularLayout(self.numOfBS, self.ISD, self.heightOfBS, self.numOfSectorsPerSite)
            self.BSLocations = self.layoutObj().T      # Initial BS locations
            self.BStracks    = self.bsRoute.nodeRoute  # BS locations across snapshots
            self.BSvelocityVector = self.bsRoute.velocityVector
            self.bsOrientation    = self.bsRoute.nodeOrientation            
            self.bsOrientation[...,0] =  self.bsOrientation[...,0] + self.layoutObj.bsOrientation

        elif self.layoutType == "Hexagonal":

            self.layoutObj   = HexagonalLayout(self.numOfBS, self.heightOfBS, self.ISD, self.numOfSectorsPerSite)
            self.BSLocations = self.layoutObj().T
            self.BStracks    = self.bsRoute.nodeRoute
            self.BSvelocityVector = self.bsRoute.velocityVector
            self.bsOrientation    = self.bsRoute.nodeOrientation
            self.bsOrientation[...,0] = self.bsOrientation[...,0] + self.layoutObj.bsOrientation

        else:
            raise ValueError("[Error-SimulationLayout]: Invalid 'layoutType' entered..!!")    
      
        if(useInitueLocations):
        #************************************************************************
        ## Placing the UEs based on the node Mobility class
            self.UELocations      = self.ueRoute.initLocation
            self.UEtracks         = self.ueRoute.nodeRoute
            self.UEvelocityVector = self.ueRoute.velocityVector
            self.ueOrientation    = self.ueRoute.nodeOrientation           
        else:
        #************************************************************************
        ## Dropping the UEs based on the drop-method
            if (self.ueDropMethod == "Rectangular"):
                self.layoutLength = layoutLength
                self.layoutWidth  = layoutWidth

                self.dropObject   = RectangularDrop(self.numOfUE, self.heightOfUE,
                                                    self.layoutLength, self.layoutWidth,
                                                    self.UEdistibution, self.UEheightDistribution)
                self.UELocations  = self.dropObject().T     # Initial UE locations
                self.UEtracks     = self.ueRoute.nodeRoute  # locations of UEs across snapshots
                self.UEvelocityVector = self.ueRoute.velocityVector
                self.ueOrientation    = self.ueRoute.nodeOrientation

            elif self.ueDropMethod == "Hexagonal":

                self.dropObject  = HexagonalDrop(self.__numBS, self.numOfUE, self.heightOfBS,
                                                 self.heightOfUE, self.ISD, self.numOfSectorsPerSite, minUEBSDistance,
                                                 self.UEdistibution, self.UEheightDistribution)
                self.UELocations = self.dropObject().T                      
                self.UEtracks    = self.ueRoute.nodeRoute   # locations of UEs across snapshots
                self.UEvelocityVector = self.ueRoute.velocityVector # vector indicating the directions of the motion of users
                self.ueOrientation    = self.ueRoute.nodeOrientation
                
            elif self.ueDropMethod == "Circular":
                self.radiusForCircularUEDrop = radiusForCircularUEDrop
                self.dropObject  = CircularDrop(self.numOfUE, self.heightOfUE, self.radiusForCircularUEDrop, 
                                                self.UEheightDistribution)
                self.UELocations = self.dropObject().T                      
                self.UEtracks    = self.ueRoute.nodeRoute   # locations of UEs across snapshots
                self.UEvelocityVector = self.ueRoute.velocityVector
                self.ueOrientation    = self.ueRoute.nodeOrientation
                
            else:
                raise ValueError("[Error-SimulationLayout]: Invalid 'ueDropMethod' entered..!!")               
              
        if(self.BStracks.shape[0] == self.UEtracks.shape[0]):
            self.__numOfSnapShots = self.UEtracks.shape[0]

        elif((self.BStracks.shape[0] == 1)):
            self.__numOfSnapShots = self.UEtracks.shape[0]
            self.BStracks         = self.BStracks.repeat(self.__numOfSnapShots, 0)
            self.BSvelocityVector = self.BSvelocityVector.repeat(self.__numOfSnapShots, 0)
            self.bsOrientation    = self.bsOrientation.repeat(self.__numOfSnapShots, axis = 0)

        elif((self.UEtracks.shape[0] == 1)):
            self.__numOfSnapShots = self.BStracks.shape[0]
            self.UEtracks         = self.UEtracks.repeat(self.__numOfSnapShots, 0)
            self.UEvelocityVector = self.UEvelocityVector.repeat(self.__numOfSnapShots, 0)
            self.bsOrientation    = self.bsOrientation.repeat(self.__numOfSnapShots, axis = 0)

        else:
            raise ValueError("[Error-SimulationLayout]: Mismatch in 'number of snapshots'..!!")                                              
          
    def __call__(self, terrain, carrierFreq, ueAntennaArray = None, bsAntennaArray = None, indoorUEfraction = 0.2, 
                 lengthOfIndoorObject = 1, widthOfIndoorObject = 1, heightOfRoom = 3, minNumberOfFloors = 1, 
                 maxNumberOfFloors = 1, forceLOS = False, clutterHeight = 5, clutterSize = 10, clutterDensity = 0.4, 
                 is3GPPBaseline = True):
        
        # The call method for Simulation Layout Class.
                
        self.terrain           = terrain        # terrain for BS-UE, UE-UE, BS-BS links
        self.carrierFrequency  = carrierFreq    # array of carrier frequencies 
        ## BS Array
        self.ueAntennaArray    = ueAntennaArray       
        self.bsAntennaArray    = bsAntennaArray     
        
        
        self.terrainBS_UE = self.terrain[0]
        self.is3GPPBaseline       = is3GPPBaseline

        self.indoorUEfraction     = indoorUEfraction    
        self.lengthOfIndoorObject = lengthOfIndoorObject
        self.widthOfIndoorObject  = widthOfIndoorObject
        self.forceLOS             = forceLOS
        self.heightOfRoom         = heightOfRoom
        
        self.compute_d3D()
        self.compute_d2D()
        
        linkLevelTerrain = np.array(["CDL", "cdl", "TDL", "tdl"])

        if self.forceLOS == True:
            
            self.linkState       = np.full((self.__numOfSnapShots, self.__numBS + self.numOfUE,
                                            self.__numBS + self.numOfUE), True, dtype = bool)
            self.indoorUEIndices = np.arange(self.numOfUE, dtype = np.uint32) if (self.terrainBS_UE[0:2] == "In") else  np.random.choice(np.arange(self.numOfUE, dtype = np.uint32), np.int32(np.round(self.indoorUEfraction*self.numOfUE)), replace = False)
        else:

            if self.terrainBS_UE == "RMa" or self.terrainBS_UE == "UMa" or self.terrainBS_UE == "UMi":
                self.computeLOSprobForOutdoorTerrain(self.compute_d2Dout())
                floorIndices   = np.random.randint(minNumberOfFloors, maxNumberOfFloors+1, 
                                                   size=self.indoorUEIndices.size, dtype = np.uint32)
                self.UELocations[self.indoorUEIndices,2] = self.UELocations[self.indoorUEIndices,2] +  floorIndices*heightOfRoom
                
                
            elif self.terrainBS_UE == "InH-OO"  or self.terrainBS_UE == "InH-MO":
                self.indoorUEIndices = np.arange(self.numOfUE, dtype = np.uint32)
                self.computeLOSprobForInH()
                
            elif  self.terrainBS_UE == "InF-SL" or self.terrainBS_UE == "InF-SH"  or \
                  self.terrainBS_UE == "InF-DL" or self.terrainBS_UE == "InF-DH"  or \
                  self.terrainBS_UE == "InF-HH": 
                
                self.clutterHeight   = clutterHeight 
                self.clutterSize     = clutterSize 
                self.clutterDensity  = clutterDensity
                self.indoorUEIndices = np.arange(self.numOfUE, dtype = np.uint32)
                self.computeLOSprobForInF()
        
            elif(self.terrainBS_UE[0:3] in linkLevelTerrain):
                if(self.terrainBS_UE[4] in np.array(["a","b","c","A","B","C"])):
                    self.linkState       = np.full((self.__numOfSnapShots, self.__numBS + self.numOfUE,
                                                    self.__numBS + self.numOfUE), False, dtype = bool)
                    self.indoorUEIndices = np.arange(self.numOfUE, dtype = np.uint32)
                    
                else:
                    self.linkState       = np.full((self.__numOfSnapShots, self.__numBS + self.numOfUE,
                                                    self.__numBS + self.numOfUE), True, dtype = bool)
                    self.indoorUEIndices = np.array([])
                
            else:
                print("Invalid BS_UE terrain passed in computing probability of LOS")

    @property
    def bsRoute(self):
        """ Route or Track or Locations of Base Station across snapshots. """
        return self.__bsRoute
    
    @bsRoute.setter
    def bsRoute(self, value):
        
        if value is None:
            self.__bsRoute = NodeMobility(None, self.__numBS, np.array([0], dtype=np.float32), None)
            self.timeInstancesBS = self.__bsRoute.timeInstances
            
        elif (isinstance(value, NodeMobility)):            
            self.timeInstancesBS = value.timeInstances
            self.__bsRoute = value           
        else:
            raise ValueError("[Error-SimulationLayout]: 'bsRoute' must be a'NodeMobility' type scalar or array!")
#--------------------------------------------------------------------------------------------------------------------------------------------------------------                   
    @property
    def ueRoute(self):
        """ Route or Track or Locations of User Equipment (UE) across snap shots."""
        return self.__ueRoute
    
    @ueRoute.setter
    def ueRoute(self, value):

        if value is None:
            self.__ueRoute     = NodeMobility(None, self.numOfUE, np.array([0],dtype=np.float32), None)
            self.timeInstances = self.__ueRoute.timeInstances
            
        elif (isinstance(value, NodeMobility)):            
            self.timeInstancesUE = value.timeInstances            
            if (np.all(self.timeInstancesBS != self.timeInstancesUE) and np.all(self.timeInstancesBS != np.array([0]))):
                raise ValueError("[Error-SimulationLayout]: timeInstances of 'ueRoute' and 'bsRoute' must match..!")
            else:
                self.timeInstances = self.timeInstancesUE                     
                self.__ueRoute = value                                
        else:
            raise ValueError("[Error-SimulationLayout]: 'ueRoute' must be a 'NodeMobility' type scalar or array!")
#---------------------------------------------------------------------------------------------------------------------------------------------------------------                  
    @property
    def bsAntennaArray(self):
        """ Object(s) of the class AntnennaArrays for each carrier frequency at BS side"""
        return self.__bsAntennaArray
    
    @bsAntennaArray.setter
    def bsAntennaArray(self, value):
        if value is None:
            self.__bsAntennaArray = np.empty(self.carrierFrequency.size, dtype=object)
            for nc in range(self.carrierFrequency.size):
                self.__bsAntennaArray[nc] = AntennaArrays("3GPP_38.901", float(self.carrierFrequency[nc]), 
                                                         np.array([1,1,1,1,1]), np.array([1,1,1,1]))
                self.__bsAntennaArray[nc]()
        elif not(isinstance(value, AntennaArrays) or (isinstance(value, np.ndarray) and np.issubdtype(value.dtype, AntennaArrays))):
            raise ValueError("[Error-SimulationLayout]: 'bsAntennaAray' must be an 'AntennaArrays' type scalar or array!")
            
        else:
            if(isinstance(value, AntennaArrays)):
                self.__bsAntennaArray = np.array([value]).repeat(self.carrierFrequency.shape[0])
                
                if not self.singleFlag:
                    print("[Warning]: Only one Antenna Array entered for multiple carrier frequencies!. Ignoring for now it might produce unexpected results.")
            else:
                if isinstance(value, np.ndarray) and np.issubdtype(value.dtype, AntennaArrays) and value.shape[0]!=self.carrierFrequency.shape[0]:
                    raise ValueError("[Error-SimulationLayout]: The num of Antenna Arrays should match the num of carrier frequencies..!")
                else: 
                    self.__bsAntennaArray = value

        """ Object(s) of the class AntnennaArrays for each carrier frequency at UE side"""
        return self.__ueAntennaArray
    
    @ueAntennaArray.setter
    def ueAntennaArray(self, value):
        
        if value is None:
            self.__ueAntennaArray = np.empty(self.carrierFrequency.size, dtype=object)
            for nc in range(self.carrierFrequency.size):
                self.__ueAntennaArray[nc] = AntennaArrays("OMNI", float(self.carrierFrequency[nc]),

    def computeLOSprobForOutdoorTerrain(self,d2Dout):
        # """
        # Computes LOS probability for outdoor scenario/terrain such as Rural Macro/"RMa", Urban Macro/"UMa" and Urban Micro/"UMi".

        # Parameters
        # ----------
        # d2Dout: NumPy.ndarray
        #     2D outdoor distance between every node pair. Dimension (numSnapShots,numNodes,numNodes), where numNodes = numBS + numUE
        # """
    
        probLOS      = np.ones(d2Dout.shape, dtype = np.float16)
        numBS        = self.__numBS
        numUE        = self.numOfUE
        numNodes     = numBS + numUE 
        d2DUEBS      = d2Dout[:,0:self.__numBS,self.__numBS:numNodes]
        numSnapShots = self.numOfSnapShots

        if self.terrainBS_UE == "RMa": # Rural Macro
            probLOS[:,0:numBS,numBS:numNodes][d2DUEBS  > 10] = np.exp((- d2DUEBS[d2DUEBS > 10]-10)*(1/1000))
            probLOS[:,numBS:numNodes,0:numBS] = np.transpose(probLOS[:,0:numBS,numBS:numNodes],(0,2,1))
            
        if self.terrainBS_UE == "UMi": # Urban Micro
            probLOS[:,0:numBS,numBS:numNodes][d2DUEBS>18] = ((1/d2DUEBS[d2DUEBS >18])*18)+(np.exp(-(1/36)*d2DUEBS[d2DUEBS >18]))*(1-((1/d2DUEBS[d2DUEBS >18])*18))
            probLOS[:,numBS:numNodes,0:numBS] = np.transpose(probLOS[:,0:numBS,numBS:numNodes],(0,2,1)) 
            
        if self.terrainBS_UE  == "UMa": # Urban Macro
            idx2          = np.where(d2DUEBS  > 18)

            ueHtLink      = (self.UELocations[:,2] + self.UEtracks[:,:,2])[:,np.newaxis,...].repeat(numBS, axis = 1)
            cPrime        = 0*ueHtLink[idx2]
            i1            = ueHtLink[idx2]>13
            cPrime[i1]    = ((ueHtLink[idx2][i1]-13)*0.1)**1.5

            probLOS[:,0:numBS,numBS:numNodes][np.where(d2DUEBS <= 18)] = 1
            probLOS[:,0:numBS,numBS:numNodes][idx2] = (18/d2DUEBS[idx2] + np.exp(-d2DUEBS[idx2]/63)*(1-18/d2DUEBS[idx2]))*(1+cPrime*1.25*(d2DUEBS[idx2]**3/1000000)*np.exp(-d2DUEBS[idx2]/150))
            probLOS[:,numBS:numNodes,0:numBS] = np.transpose(probLOS[:,0:numBS,numBS:numNodes],(0,2,1)) 

        ## Setting link State based on Probability of LoS    

        self.linkState  = np.full(probLOS.shape, False, dtype = bool)
        
        x          = np.random.rand(probLOS.shape[0],probLOS.shape[1],probLOS.shape[2])
        xTop       = x[:,0:numBS,numBS:numNodes]
        
        self.linkState[:,0:numBS,numBS:numNodes][probLOS[:,0:numBS,numBS:numNodes]>=xTop] = True
        self.linkState[:,numBS:numNodes,0:numBS] = np.transpose(self.linkState[:,0:numBS,numBS:numNodes],(0,2,1))

        diagIdx    = tuple(np.array(np.concatenate(
            (np.array(np.repeat(np.asmatrix(np.arange(0,numSnapShots)),numNodes,axis=1)),
             np.matlib.repmat(np.asmatrix(np.arange(0,numNodes)),1,numSnapShots),
             np.matlib.repmat(np.asmatrix(np.arange(0,numNodes)),1,numSnapShots)),axis=0)))
        
        self.linkState[diagIdx] = True
