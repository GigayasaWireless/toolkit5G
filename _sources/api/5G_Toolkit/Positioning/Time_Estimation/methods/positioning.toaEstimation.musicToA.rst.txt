MUSIC based ToA Estimation
======================================
This module implements a method for estimating the delays using **MU**\ ltiple **SI**\ gnal **C**\ lassification (MUSIC)
method.

------------

**Code example**

.. code-block:: Python

    # shape of Hf: (numSubcarriers, numObservations)
    L         = 12
    min_delay = 10**(-8)
    max_delay = 2.5*10**(-6)
    music     = MUSIC_ToA()
    delays    = music(Hf, numberOfPath=L, subCarrierSpacing=scs, min_delay=min_delay, max_delay=max_delay)
