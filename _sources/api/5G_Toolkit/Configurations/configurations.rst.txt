5G Configurations
=================
This package provides modules for generating configurations for various physical channels, 5G procedures and other
features. In 5G, few of the implementations expects tens of parameters to be configured. These configuration provide
very simple APIs for generating all these parameters. The users can input the few parameters of their interest and
configurations will selects the remaining parameter randomly from the set of legit possible configurations.
