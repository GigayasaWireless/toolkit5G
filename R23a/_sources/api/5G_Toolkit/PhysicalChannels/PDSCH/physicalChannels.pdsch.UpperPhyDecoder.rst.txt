PDSCH: Upper Physical layer Chain Decoder
#########################################
This module implements the physical downlink shared channel.

.. image:: ../../../figures/PhysicalChannels/PDSCHUpperPhyDecoder.png
  :width: 800
  :alt: Alternative text

------------

The following code block illustrates how to initialize and invoke PDSCH Upper Phy Decoder

.. code-block:: python

    numSymbols         = 13
    numRB              = 25
    mcsIndex           = 2
    numlayers          = [1]
    scalingField       = '00'
    additionalOverhead = 0
    dmrsREs            = 3
    numTBs             = 1
    mcsTable           = 'pdschTable2'


    pdschDecoderUpperPhyChain = PDSCHDecoderUpperPhy(numSymbols,numRB,mcsIndex,numlayer,scalingfield,additionalOverhead,dmrsREs,numTBs,mcsTable)
    #LLRs  = An array of estimated likelihood ratios (LLRs)of size (...,numTargetBits)

    DecodedBits               = pdschDecoderUpperPhyChain()

------------
