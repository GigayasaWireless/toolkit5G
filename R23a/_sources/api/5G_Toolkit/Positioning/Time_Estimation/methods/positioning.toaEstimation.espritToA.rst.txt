ESPRIT based ToA Estimation
=============================================================================
This module implements a method for estimating the delays using Estimation of Signal Parameters via Rotational Invariance
Techniques (ESPRIT) method.

------------

**Code example**

.. code-block:: Python

    # shape of Hf: (numSubcarriers, numObservations)
    L         = 12
    esprit    = ESPRIT_ToA(Hf.shape)
    delays    = esprit(Hf, numberOfPath=L, subcarrierSpacing=scs)
