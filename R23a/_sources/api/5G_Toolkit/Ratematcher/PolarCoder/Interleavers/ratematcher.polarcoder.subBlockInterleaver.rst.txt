Sub Block Interleaver for Polar Coder
=====================================
The sub-block interleaver divides the encoded bits into sub-blocks of size 32 and these sub-blocks are interleaved before
bit-selection.

.. note::
    The sub-block interleaver divides the input blocks into sub-blocks of size 32 each.
