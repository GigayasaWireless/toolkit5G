Install 5G Toolkit
===================

This page takes the user through the complete process of installing the 5G Toolkit. It also provides a detailed installation 
procedure for new users.

How to install 5G Toolkit:
--------------------------

5G Toolkit can be installed easily using pip which comes pre-installed with Python-3. The details of the installation are 
provided below. The modules and libraries of 5G Toolkit are written in Python and is compatible with all Python versions >= 3.7.4.

System Requirements:
--------------------

.. list-table:: system Requirements
    :widths: 25 25
    :header-rows: 1

    * - Subject
      - Version
    * - Architecture
      - x86_64
    * - Python
      - \>= 3.7.4 and < 3.12

------------------------

.. Warning::

    The users with any other operating system and architecture can request the package at toolkit5g@gigayasa.com We will create a new package for the requested configurations and provide installation procedure on this page itself.

Install 5G Toolkit on Ubuntu (20.04/22.04) and Windows (10/11)
---------------------------------------------------------------

If you are using pip for installation of packages, then run following in your terminal in your target environment.

The **template command** for installation is given below:

.. image:: _static/Install_Toolkit.png

.. note::

    "--no-cache-dir" flag is used to block pip from using cached version of the package for installation. Users can remove it if not relevant for them.

**For Python 3.7**:

.. note::

    Initial version of Python 3.7 is buggy. Hence, only Python>=3.7.4 is supported.

.. code-block:: bash

    pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_7/toolkit5G-0.10-py3-none-any.whl


**For Python 3.8**:

.. code-block:: bash

    pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_8/toolkit5G-0.10-py3-none-any.whl


**For Python 3.9**:

.. code-block:: bash

    pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_9/toolkit5G-0.10-py3-none-any.whl


**For Python 3.10**:


.. code-block:: bash

    pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_10/toolkit5G-0.10-py3-none-any.whl


**For Python 3.11**:

.. code-block:: bash

    pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_11/toolkit5G-0.10-py3-none-any.whl

------------

To confirm installation of the library, you can check the installation library:

.. code-block:: bash

    pip show toolkit5G

------------

Detailed Installation Procedure
--------------------------------

This section covers detailed installation procedure for Windows (10 and 11), Ubuntu (20.04 and 22.04). It covers following
steps
 #. Install `Conda <https://docs.conda.io/en/latest/miniconda.html>`_
 #. `Create Virtual environment <https://docs.python.org/3/library/venv.html>`_ and install Python in it.
 #. Install Jupyter Notebook
 #. Install 5G Toolkit
 #. Import and use Toolkit in Jupyter Notebook.

Please `Click Here <detailedInstall.html>`_  for detailed installation procedure.

License 5G Toolkit
------------------

The 5G toolkit requires an active license for usage. The new users can request a one month trial license for 5G Toolkit
through this link [Request Trial License](`Request Trial <https://gigayasa.com/request-trial/>`_). Furthermore, you can purchase an annual
or perpetual license [Purchase License](`Purchase License <https://gigayasa.com/request-pricing/>`_).

Activate the 5G Toolkit License
-------------------------------

 #. Download the **Central-Hub** shared with you over the email.
 #. Unzip and run the executable file.
 #. Once the installation is complete, open the **Central-Hub** from your applications.
 #. Login if you have the Gigayasa account, otherwise:
   #. Create an account with Gigayasa.
   #. Verify your email
   #. Log into the Central-Hub.
 #. Now fetch the License key shared with you over the email.
 #. Activate your License.
 #. If activation is successful, please close the Central-Hub.
 #. You are good to go please open the jupyter Notebook and try to "import toolkit5G as tk"
