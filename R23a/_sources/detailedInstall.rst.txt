Install 5G Toolkit
==================

This page takes the user through the complete process of installing all the packages to the point where they can use the
5G toolkit easily in a Jupyter Notebook.

System Requirements:
--------------------

.. list-table::
    :widths: 25 25
    :header-rows: 1

    * - Subject
      - Version
    * - Architecture
      - x86_64
    * - Python
      - \>= 3.7.4 and < 3.12

.. tip::

    The users with any other operating system and architecture can request the package at toolkit5g@gigayasa.com We will
    create a new package for the requested configurations and provide installation procedure on this page itself.


Dependent Libraries:
--------------------

.. list-table::
    :widths: 25 25
    :header-rows: 1

    * - Library
      - Version
    * - NumPy
      - > 1.16
    * - SciPy
      - >=1.6.0
    * - matplotlib
      - > 3.4
    * - Sionna
      - <=0.13 (For version 0.14 LLVM may create problem)


Install Miniconda
------------------
If Anaconda is already installed on your system, then you can skip this step. The following instructions install the
Miniconda which is a lighter version of Anaconda. First we will download miniconda using the command:

.. note::

    Please install the latest version of the installer from `this link <https://docs.conda.io/en/latest/miniconda.html>`_ .
    Following links might be outdated.


.. tabs::

   .. group-tab:: Linux

        Lets start by updating the system:

        .. code-block:: bash

            sudo apt-get update

        Open the terminal, (Shortcut CTRL+ALT+T) and execute the following command in it:

        .. code-block:: bash

            wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

        Followed by, installing miniconda using the downloaded package.

        .. code-block:: bash

            bash Miniconda3-latest-Linux-x86_64.sh

        Once the installation is completed, we can check the version installed Conda on our system.

        .. code-block:: bash

            conda --v

   .. group-tab:: Windows

        1. Download the Miniconda installer from this link:

           * For 32 bit version of Windows: `Download Miniconda Installer <https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86.exe>`_
           * For 64 bit version of Windows: `Download Miniconda Installer <https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86_64.exe>`_

        2. Install **Miniconda** by executing this installer.

        Click on start button and find Anaconda Prompt (**Not Anaconda bash**). Once the installation is completed, we can check the Conda
        version installed on our system:

        .. code-block:: bash

            conda --v

Now, we will use conda to create a new environment for python 3.11 (this command will automatically install python 3.11
in the target environment):

.. code-block:: bash

    conda create -n gigayasa python==3.11


Finally, we will activate the created environment by executing

.. code-block:: bash

    conda activate gigayasa

.. important::
    We will install everything from further now in this environment only.


Install Jupyter Notebook
-------------------------

Install Jupyter for interactive scripting in Python. Execute the following command in the terminal in the
created environment.

.. code-block:: bash

    pip install Jupyter


Install 5G Toolkit
------------------

Finally, we are ready to install 5G Toolkit:

.. tabs::

   .. group-tab:: **Python 3.7**

        .. code-block:: bash

            pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_7/toolkit5G-0.11-py3-none-any.whl

   .. group-tab:: **Python 3.8**

        .. code-block:: bash

            pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_8/toolkit5G-0.11-py3-none-any.whl

   .. group-tab:: **Python 3.9**

        .. code-block:: bash

            pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_9/toolkit5G-0.11-py3-none-any.whl

   .. group-tab:: **Python 3.10**

        .. code-block:: bash

            pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_10/toolkit5G-0.11-py3-none-any.whl

   .. group-tab:: **Python 3.11**

        .. code-block:: bash

            pip install --no-cache-dir https://github.com/GigayasaWireless/toolkit5G/raw/R23a.0.11/package/python3_11/toolkit5G-0.11-py3-none-any.whl



To confirm installation of the library, you can check the installation library:

.. code-block:: bash

    pip show toolkit5G

Final Confirmation
-------------------

The installation will be truly successful if we are able to import the toolkit5G packages. We can verify it using 
**Jupyter Notebook**, open the jupyter notebook using following command in your browser.

.. code-block:: bash
    jupyter notebook


Create a new jupyter notebook and run the following lines of code:

.. code-block:: ipython3

    import toolkit5G as tk
    tk.__version__

The toolkit is successfully installed if it shows the toolkit5G version as shown below.

.. parsed-literal::

    'R23a.0.10'

Otherwise, something went wrong. If it shows "License Error", then you will have to
`activate the License <#activate-the-5g-toolkit-license>`_. But still if you are struggling please email us at any of
the following emails:

#. `<toolkit5g@gigayasa.com>`_
#. `<support@gigayasa.com>`_
#. `<vikram@gigayasa.com>`_

We will be more than happy to help you in installation.


License 5G Toolkit
------------------

The 5G toolkit requires an active license for usage. The new users can request a one month trial license for 5G Toolkit
through this link `Request Trial <https://gigayasa.com/request-trial/>`_. Furthermore, you can purchase an annual
or perpetual license `Purchase License <https://gigayasa.com/request-pricing/>`_.

Activate the 5G Toolkit License
-------------------------------

 #. Download the **Central-Hub** shared with you over the email.
 #. Unzip and run the executable file.
 #. Once the installation is complete, open the **Central-Hub** from your applications.
 #. Login if you have the Gigayasa account, otherwise:
   #. Create an account with Gigayasa.
   #. Verify your email
   #. Log into the Central-Hub.
 #. Now fetch the License key shared with you over the email.
 #. Activate your License.
 #. If activation is successful, please close the Central-Hub.
 #. You are good to go please open the jupyter Notebook and try to "import toolkit5G as tk"
